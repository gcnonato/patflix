Suggested Videos

This is a collection of videos that I love watching and recommend many of my friends to see. Many of these videos are thought-provoking, inspiring, or educational, and I hope they connect with you in a similar fashion as they did with me.

Everything you see below this section was dynamically generated by JavaScript that I wrote from scratch- no jQuery, no plugins, no copy-paste. It uses a static JSON file as the source for all video information, and the sections are dynamically generated based on category tags that the videos contain. (A function takes a category as a parameter, searches the JSON for videos that fall under the category, and then adds them to an Array of objects). The JavaScript builds the elements and then appends them to the DOM.
